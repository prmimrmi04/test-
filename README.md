# Agent Repository

This repository contains tools and tasks for AI agents.

## Structure
- `agent_tasks.json`: A list of tasks for agents to perform.
- `scripts/`: Scripts to manage tasks and agents.
  - `list_active_tasks.py`: A Python script to list all active tasks (status 'todo') from `agent_tasks.json`.
  - `list_completed_tasks.py`: A Python script to list all completed tasks (status 'done') from `agent_tasks.json`.
  - `validate_agent_tasks.py`: A Python script to validate the structure of `agent_tasks.json`.
- `tests/`: Tests for the scripts.

## Tasks
See `agent_tasks.json` for active and completed tasks.
